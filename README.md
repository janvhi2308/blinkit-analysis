## 📊 Project Overview

This project is an end-to-end **Sales Analysis Dashboard** built in **Power BI** for **Blinkit** — India's Last Minute Delivery App. The dashboard provides deep insights into sales performance, outlet distribution, item types, and customer ratings across different tiers of outlets.

## 🎯 Key Metrics Tracked

| Metric | Value |
|--------|-------|
| 💰 Total Sales | **$1.20M** |
| 📦 No. of Items | **8,523** |
| 📈 Avg Sales | **$141** |
| ⭐ Avg Rating | **4.0** |

## 📌 Dashboard Features

- **Fat Content Analysis** — Donut chart comparing Low Fat vs Regular items by total sales
- **Item Type Breakdown** — Bar chart of sales across 16+ product categories
- **Fat by Outlet** — Tier-wise fat content distribution (Tier 1, 2, 3)
- **Outlet Establishment Trend** — Line chart showing sales trends from 2012–2022
- **Outlet Size Distribution** — Donut chart (Small / Medium / High)
- **Outlet Location Analysis** — Horizontal bar chart by Tier
- **Outlet Type Summary Table** — Detailed matrix with sales, items, avg rating & item visibility

## 🔍 Filters / Slicers

- **Outlet Location Type** (Tier 1 / Tier 2 / Tier 3)
- **Outlet Size** (Small / Medium / High)
- **Item Type** (16+ categories)

## 🚀 How to Use

1. **Clone** this repository:
   ```bash
   git clone https://github.com/janvhi2308/blinkit-analysis.git
   ```

2. **Open** `Blinkit Analysis.pbix` in [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download)

3. **Interact** with the slicers to explore data by outlet location, size, and item type

4. **Publish** to Power BI Service for sharing with your team (requires Power BI Pro or Premium)

## 📈 Insights Uncovered

- **Supermarket Type 1** leads with **$787.55K** in total sales and **5,577 items**
- **Tier 3** outlets generate the highest sales at **$472.13K**
- **Fruits & Snack Foods** are the top-selling item categories (~$0.18M each)
- Sales peaked in **2018** at **$205K** outlet establishment
- **Medium-sized** outlets account for the largest share of outlet size revenue

## 📁 Data Source

The dataset contains outlet-level retail data from Blinkit including:
- Item details (type, fat content, visibility, MRP)
- Outlet details (location, size, type, establishment year)
- Sales performance metrics and customer ratings
